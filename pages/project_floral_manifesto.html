Project Links

Floral Manifesto is a blog written by Margeuax Feldman, a PhD Candidate in English and Sexual Diversity Studies at the University of Toronto.

The goal of this project was to re-design and "re-brand" the authors original wordpress.com blog. The blog had expanded in popularity and needed not only a new look, but new features to further improve the quality of the "Floral Manifesto" brand.

Developing this Wordpress based blog required me to build a Wordpress theme from scratch, as the wire frame for the new website design was developed by a graphic designer. There are three custom-post-types which are easily accessable for the author in the Wordpress Dashboard. Each custom-post-type is accompanied by meta-data which is used to organise the posts content within their respective pages.

For example, follow this [ GitHub ] link to view a Wordpress Query I built which handles the custom-post-types for the [ "Reading List" ] page.
